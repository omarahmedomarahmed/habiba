## The Website Is Not a Website

Most founders think: **Website = Information**

Wrong.

The website exists to:
- **Convert**
- **Educate**
- **Build Trust**
- **Capture Demand**
- **Generate Revenue**

For 24Therapy, the website has 3 completely different audiences:

| Audience | Identity |
|----------|----------|
| Audience 1 | People looking for help (Patients) |
| Audience 2 | Mental health professionals (Therapists) |
| Audience 3 | Organizations (Practices, clinics, enterprises) |

The website must speak to all three.

## Visual Direction

**Style:** Modern Silicon Valley healthcare company.

**Inspiration:** Enterprise healthcare software meets consumer mental wellness.

**Visual language:**
- Soft gradients
- Navy foundation
- Bright blue accents
- White space
- Rounded corners
- Large typography
- Minimal clutter

### Color System

| Role | Color |
|------|-------|
| Primary | Deep Navy (`#0A2342`) |
| Secondary | Healthcare Blue (`#1F5EFF`) |
| Accent | Teal/Cyan (`#24C8DB`) |
| Neutral | White / Gray Scale |
| Alert | Amber |
| Error | Red |
| Success | Green |

### Typography

- Headings: Modern sans-serif (Inter)
- Body: Highly readable
- Large spacing
- Accessible (WCAG compliant)
