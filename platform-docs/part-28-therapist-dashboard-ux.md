## Key Screens (Outlined)

### Dashboard
- Today's sessions widget
- Patient alerts widget
- Radar activity widget
- Revenue this month widget
- Upcoming assessments widget
- Recent reports widget

### Patient Management
- Patient list (sortable, filterable, searchable)
- Patient card hover states
- Create patient modal/drawer
- Patient profile (10+ tabs: Overview, Sessions, Notes, Assessments, Medications, Goals, Timeline, Files, Reports, Memory)

### Session Room
- Left panel: Transcript (realtime)
- Center: Video/audio
- Right panel: AI Copilot suggestions
- Bottom bar: Session controls
- Floating: Note pad

### Post-Session Workflow
- Review transcript
- Review AI note draft
- Edit note (SOAP/DAP/BIRP)
- Generate report
- Update memory
- Schedule follow-up
